# Travel Agency Platform - Frontend

Next.js frontend for travel agency platform with multi-language support.

## Setup

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Environment Variables

Create `.env.local`:

```
NEXT_PUBLIC_API_URL=http://localhost:8000/api
NEXT_PUBLIC_SITE_URL=http://localhost:3000
NEXT_PUBLIC_WHATSAPP_NUMBER=51987654321
```

## Features

- ✅ Multi-language (ES, EN, PT)
- ✅ SEO optimized
- ✅ Responsive design
- ✅ WhatsApp integration
- ✅ Booking forms
